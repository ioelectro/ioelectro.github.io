<p>
    فوجیتسو طیف کاملی از ابزارهای نرم افزاری از جمله کامپایلر ANSI C با کد بسیار بهینه، ماکرو اسمبلر، شبیه ساز و غیره را ارائه می دهد.<br>
    برای تسهیل کار، این محصولات در یک مجموعه واحد به نام "Softune" ترکیب می شوند که به شما امکان تولید کامپایل خودکار، شامل قابلیت های اشکال زدایی فراوان و سیستم کمک داخلی را می دهد.<br>
    مجموعه اصلی نرم افزار برای میکروکنترلرهای F2MC-8L/16L/16LX شامل عملکردهای زیر است:
</p>

<ul class="en">
    <li>Simple menu system with "mouse" management</li>
    <li>Support for collaboration on a multi-user project</li>
    <li>Automatic start of the editor, compiler, assembler, and debugger</li>
    <li>Moving to Error by Simply Pressing a Mouse</li>
    <li>Using a built-in editor with spelling check, or external editor</li>
    <li>Built-in guide to the compiler and assembler</li>
    <li>Configuring the user's working environment</li>
    <li>Preservation/rest recovery of the working environment </li>
</ul>

<p>امکانات دیباگر:</p>

<ul class="en">
    <li>Debugging in source texts (C, Assembler or mixed)</li>
    <li>Navigation Features: Go, Go to line, Step out, Step over, Call function, Reset</li>
    <li>Browsing Variables: Automatic Update (x Sec), formatting (Hex, Bin, etc.), structures</li>
    <li>View Global and Local Variables</li>
    <li>Memory Window : Formatting, Symbolic Debugging, Code Debugging, ASCII</li>
    <li>Disassembler Window</li>
    <li>Simulator : debugging of I/O operations with data storage, interrupt simulation</li>
    <li>Service functions: log files and command files. </li>
</ul>

<p>امکانات شبیه ساز:</p>
<ul class="en">
    <li>Support for low-energy microcontrollers</li>
    <li>Debugging in real time</li>
    <li>Multi-level event controller</li>
    <li>Viewing the contents of memory "on the fly"</li>
    <li>Additional cables for connection in the debuggable system</li>
</ul>
